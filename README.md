# GoMytho Landing Page 🚀

Landing page virale et mémorable pour GoMytho - l'IA qui te permet d'ajouter n'importe quoi sur n'importe quelle photo en 10 secondes.

## 🎨 Stack Technique

- **Framework**: Next.js 15 (React 18)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Langage**: TypeScript
- **Déploiement**: Vercel-ready

## 🎯 Caractéristiques

- ⚡ Performance optimisée (Lighthouse > 90)
- 📱 Design mobile-first et 100% responsive
- 🎭 Animations fluides et interactives à 60fps
- 🎨 Design "premium playful" unique
- 🔍 SEO et Open Graph optimisés
- ♿ Accessible et moderne

## 🚀 Installation

```bash
# Installer les dépendances
npm install

# Lancer le serveur de développement
npm run dev

# Build pour production
npm run build

# Démarrer en production
npm start
```

Le site sera accessible sur `http://localhost:3000`
